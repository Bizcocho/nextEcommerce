## Getting Started

The back-end server should be running:
https://github.com/Bizcocho/nextEcommerceBackend


Then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
